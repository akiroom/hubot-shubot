hubot-shubot
============

> 人もテニスも、ラブから始まる

Installation
------------

* Run the `npm install` command.

```
$ npm install hubot-shubot -save
```

* Add the following code in your `external-scripts.json` file.

```
["hubot-shubot"]
```

Usage
-----

```
Hubot> つらい
気にすんなよ！くよくよすんなよ！大丈夫、どうにかなるって！ドントウォーリー！ビーハッピー！
```
